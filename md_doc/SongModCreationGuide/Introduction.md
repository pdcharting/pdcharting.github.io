This guide is very long and drawn out. Modding any *Project DIVA* game is a long and tedious process and *Mega Mix+* is no different. **Take your time, thoroughly read what is written, and take things step-by-step if you need to.** The guide will provide important context as to what you’re doing and why you’re doing it. **If you’re unable to do that, modding Project DIVA probably isn’t for you.**

This guide was originally written in January of 2023, but was rewritten in October of 2025. If you are experienced and want to leave feedback on ways to improve the guide, feel free to leave a comment (where applicable) or ping me (@lavverso) in the [Mega Mix+ Modding Discord server][pdmm_discord], a server I made to provide assistance for modding *Hatsune MIku: Project DIVA Mega Mix+* specifically. You can also join and ask for help if you have any technical issues.

## What You’ll Learn<!-- {docsify-ignore} -->
This guide will help you make a mod with an Arcade chart, accompanied by a 2D PV chart. It will function similarly to in-game songs like *Pink Moon*, *Song of Eternity -DIVAMIX-*, *Dreamin Chuchu*, or *JITTERBUG*.

## What You Won’t Learn<!-- {docsify-ignore} -->
This guide is <ins>not</ins> a tutorial for how to create charts. It is a guide on how to set them up as a mod in *Hatsune Miku: Project DIVA Mega Mix+*.
