### 你是谁？为什么要制作这个文档？
笔者在国内被人称为<mark>安定</mark>，你也可以称笔者为<mark>hiki8man</mark>  
笔者在自己短暂的五年写谱生涯里写过一篇国内最早的Mega39+自制谱制作教程，也简单介绍过制作自制谱的一些基规  
但受限于笔者当时自身的写谱能力与时代变迁，旧有的教程已经过时，笔者逐渐萌生了重新编写文档的想法  

### 为什么不继续在B站专栏和语雀上继续更新教程？
笔者最初的文档教程发布在B站专栏上，这之后专栏逐渐被B站给淡化，寻找起来比较麻烦，于是笔者后来在经过一番寻找后选择了语雀继续编写修订文档。  
语雀本身十分实用，尽管其发布需要付费，笔者认为语雀的功能性完全匹配其价格。基于语雀又编写了上万字的教程  
在那之后，笔者发现使用文字描述<mark>音符图标</mark>不是很直观，而语雀并不能插入自定义图标  
一番兜兜转转后，笔者最终选择了docsify配合魔改的emoji插件重新编写这个文档

### 我完成了谱面就算是谱师吗？  
**<font color=red>不是</font>**  
笔者曾经在知乎上看到过一个很有意思的解释：
* <mark>**谱师**</mark>会对自己有较高的要求，慢慢思考如何让谱面表达出自己想要表达的东西，认真听取人们的意见，精益求精，从自己制谱能力的进步中得到成就感
* <mark>**制谱人**</mark>本身只是将制作谱面当作一个娱乐，谱面质量是可选项，只要谱面完成了得到了玩家的认同就足够满足  
至于正在阅读这个文档的各位究竟是想要成为谱师还是制谱人，这就不是笔者能掌控的了

### 我打不了Ex难度，能够尝试制作自制谱吗？  
**<font color=red>不建议</font>**  
谱面就像是画布上的画，谱面中的音符则为画上的各种颜色。**如果你连每种颜色适合使用在什么场景下都不知道，那又如何能绘制出一个美丽的画呢？**  
国内很多热门音游的自制谱社区**需要玩家有一定打歌水平**才能加入。尽管这么做的确能够一定程度上解决乱写谱的问题，但也**让社区变得更加封闭**  
**笔者并不希望Diva自制谱社区也变成这样**，希望各位制谱人在制作谱面前先熟悉游戏，熟悉一下画笔染料后再拿起手中的画笔创作谱面

### 我害怕自己制作的谱面很烂怎么办？  
**千里之行，始于足下**  
如果你连第一次的尝试都没有，怎么能知道自己以后能不能学会制作谱面呢？  
即使你的初谱做的再怎么差，你也通过制作谱面这个过程学会了谱面编辑器的基础使用。**无论多小的进步也是进步**，只要你愿意坚持，这些小的进步会慢慢累积然后蜕变。**请相信自己一定能成功**
